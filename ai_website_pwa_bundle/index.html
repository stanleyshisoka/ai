Guidelines: How the AI Works

  • The AI runs as a Python desktop application (Tkinter interface with voice/text features).
  • It includes: login system, password vault, encryption, voice assistant, reminders, and security utilities.
  • Since it uses desktop libraries (Tkinter, speech recognition, etc.), it must be run on a local computer ‚Äî not inside a browser.

Requirements to Run

  • Python 3.9+ installed on your computer.
  • Required libraries: tkinter (built-in), speechrecognition, pyttsx3, cryptography, pyaudio, and requests.
  • Install dependencies using: pip install speechrecognition pyttsx3 cryptography pyaudio requests
  • Run with: python AI_assistant_enhanced_secure.py
